# dokumentu-sudarymas
Praktikų dokumentų generavimo programa.
Ši programa buvo sukurta praktikos metu studijų skyriui, siekiant automatizuoti ir palengvinti studentų praktikos dokumentų generavimo procesą.
Repozitorijoje nėra šablonų ir kitų konfidencialių failų (kaip duomenų bazė), kurie priklauso studijų skyriui.
Dėl šios priežasties programa neveiks (ir neturėtų veikti) pagal paskirtį.
